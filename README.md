## 🤖 **Introdução**

Este projeto é uma aplicação **Full Stack** desenvolvida com **Next.js 14**, inspirada no conceito do **Threads**, com o propósito de **estudo e aprofundamento em tecnologias modernas** de frontend e backend.

A ideia é recriar uma plataforma social completa, com autenticação, comunidades, comentários aninhados e gerenciamento de usuários — tudo integrado a um backend escalável e com foco em boas práticas de código.

O foco principal é entender **arquitetura de sistemas modernos**, **integração entre serviços**, **SSR com Next.js**, **MongoDB com schemas complexos** e **validação robusta de dados com Zod**.

---

## ⚙️ **Stack Tecnológica**

- **Next.js 14** – Framework React com suporte a SSR, rotas dinâmicas e otimização de performance.
- **TypeScript** – Tipagem estática para segurança e previsibilidade.
- **MongoDB** – Banco de dados NoSQL para estruturas flexíveis e escaláveis.
- **Clerk** – Autenticação moderna com suporte a login via email e redes sociais.
- **TailwindCSS** – Estilização responsiva e ágil.
- **Shadcn UI** – Componentes acessíveis e personalizáveis.
- **Zod** – Validação de dados no backend e frontend.
- **React Hook Form** – Gerenciamento eficiente de formulários.
- **UploadThing** – Upload simplificado de arquivos.
- **Webhooks e APIs Serverless** – Comunicação em tempo real e arquitetura desacoplada.

---

## 🔋 **Funcionalidades**

✅ **Autenticação Completa:** Login com Clerk (Google, GitHub e email), com gestão de perfil.  
✅ **Feed Dinâmico:** Página inicial com threads mais recentes e responsividade total.  
✅ **Criação de Threads:** Sistema de postagens com texto e mídia.  
✅ **Comentários Aninhados:** Estrutura hierárquica para discussões encadeadas.  
✅ **Comunidades:** Criação, gerenciamento e busca de comunidades com paginação.  
✅ **Notificações de Atividade:** Alertas quando alguém comenta ou interage.  
✅ **Perfis de Usuário:** Exibição pública de informações e ajustes de perfil.  
✅ **Administração de Comunidades:** Alteração de funções, remoção de membros e posts exclusivos de admins.  
✅ **Busca em Tempo Real:** Filtro de usuários e comunidades com SSR otimizado.  
✅ **Validação com Zod:** Controle de integridade e consistência dos dados.  
✅ **Upload de Imagens:** Feito via UploadThing com integração segura.  
✅ **Performance e SEO:** SSR com renderização instantânea e otimização de metadados.
